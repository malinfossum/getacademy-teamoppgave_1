# Teamoppgave 1 – Inventory Filler

Dette prosjektet er en teamoppgave fra GET Academy.  
Oppgaven går ut på å lage en enkel *inventory*-løsning ved hjelp av HTML, CSS og JavaScript.

## Beskrivelse
Brukeren kan klikke på ulike elementer for å legge dem i et inventory, tilsvarende et inventory i et spill.  
Inventory lagres i en variabel og kan vises/skjules i grensesnittet.

Prosjektet er implementert i én fil (`index.html`) og fokuserer på grunnleggende JavaScript-konsepter.

## Funksjonalitet
- Klikkbare elementer som legges i inventory
- Inventory lagres i en variabel (array)
- Funksjon med parameter og return
- Mulighet for å vise/skjule inventory
- Enkel visning av innholdet i inventory

## Teknologi
- HTML
- CSS
- JavaScript

## Team
